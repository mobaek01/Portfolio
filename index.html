M.E.V. Calorie Counter

Postgres

Django

React

Node.js

    DESCRIPTION

  • Built within a group of three, M.E.V. Calorie Counter is an application that allows users to log the foods and its calories.
  • The application uses basic JavaScript functionality to calculate the total calories consumed from the point of the first logged food/calorie to the last. Also using the React library, Chart.js, we were able to display a bar graph that displays the calories logged per day.
  • M.E.V. Calorie Counter has two search functionalities. The first functionality allows users to search from the Back-End API and either re-log the same information or partially use the information and edit other fields. The second functionality allows users to search a third-party API CalorieNinjas that would provide the calorie count of the food that they searched.
  • The Back-End was built using Postgres and Django and the Front-End was built using React and Node.js.
